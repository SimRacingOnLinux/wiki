# BeamNG.Drive

BeamNG.Drive ships with a native Linux build.

## BeamMP

The BeamMP launcher supports Linux but they don't release precomplied Linux binaries.
Instructions on how to build the launcher can be found [in the BeamMP docs](https://docs.beammp.com/game/getting-started/?h=linux#2b-linux-installation). For Arch users a package exists on AUR, [beammp-launcher-git](https://aur.archlinux.org/packages/beammp-launcher-git).
