# Sim Racing on Linux

Sim Racing on Linux is a community-maintained repository of information for getting racing simulators and related hardware working on Linux.

The goal of this wiki is to collect practical documentation on:

- sim setup on Linux
- wheel, pedal, shifter, and button box support
- force feedback configuration
- compatibility notes for specific games and devices
- troubleshooting guides and workarounds

See the [Contributing](contributing.md) page for information on editing this wiki.
